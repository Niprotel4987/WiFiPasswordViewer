# 📶 WiFi Password Viewer

Um aplicativo desenvolvido em **Python** para visualizar informações de redes Wi-Fi salvas no Windows de forma simples, organizada e moderna.

> 🚧 **Projeto em desenvolvimento**
>
> Este projeto ainda está em constante evolução. Novas funcionalidades serão adicionadas nas próximas versões.

---

## ✨ Funcionalidades atuais

- Interface gráfica moderna
- Listagem de redes Wi-Fi salvas
- Exportação para arquivo TXT
- Exportação para arquivo CSV
- Estrutura organizada em módulos
- Projeto versionado com Git e GitHub

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/Niprotel4987/WiFiPasswordViewer.git
```

Entre na pasta:

```bash
cd WiFiPasswordViewer
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

Execute o projeto:

```bash
python main.py
```
